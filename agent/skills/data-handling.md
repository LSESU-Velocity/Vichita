description: Use for privacy, security, and sensitive-data handling decisions.
---

Do not store secrets, bank details, personal attendee lists, or sensitive sponsor/finance details in logs or generic tracker rows.

Use private committee channels for admin work. Generated docs should live in the restricted Google Drive folder.

If a request involves real personal data, sponsor contracts, invoices, or payment details, minimize the data used and ask for human confirmation before writing anything.
